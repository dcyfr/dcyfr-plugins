# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in a plugin listed here, please **do not** open a public issue.

Instead:
- Email **security@dcyfr.ai** with subject `[PLUGIN VULN] <plugin-name>`
- Include: plugin name, version, description, and steps to reproduce
- We will acknowledge within 24 hours and resolve within 14 days

## Plugin Removal Policy

Plugins with unaddressed critical vulnerabilities will be removed after 7 days of no response from the author.
